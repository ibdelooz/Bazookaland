# Bazookaland — Site Web

Site vitrine pour Bazookaland, événement culturel à Lomé, Togo (Akaf Family).

## Lancer le projet en local

**Prérequis :** Node.js

1. Installer les dépendances :
   `npm install`

2. Lancer le serveur de développement :
   `npm run dev`

3. Ouvrir le site sur `http://localhost:3000`

## Build de production

`npm run build`

Le résultat est généré dans le dossier `dist/`, prêt à être déployé (Netlify, Vercel, etc.).
